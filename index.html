HillCraft Rules
  1. Common sense
    Treat others how you want to be treated.
  2. Griefing
    Griefing, pranks, and PVP are only allowed if you have permission from the target, and nobody else is affected.
  3. Hacking
    Hacked clients are forbidden, X-Ray resource packs included.
  4. Keep it school appropriate
    To keep the server friendly for students of all ages, please be respectful to everyone. If you wouldn't say/do it in school, don't say/do it here. Do not attempt to bypass our chat filter.
  5. Glitches
    The only allowed glitches are:
    • TNT Dupe
    • Wither cages
    • Going above the nether roof
    • Breaking bedrock
